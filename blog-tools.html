Chaos SNI Finder

The flagship tool from Ethan Chaos. Automatically discovers and tests SNI hosts that work on your current carrier network — producing ranked results ready for use in HTTP Injector and HA Tunnel Plus configs.

  • Automated SNI host discovery & testing
  • Ranks results by latency and reliability
  • Generates ready-to-use config parameters
  • Works with HTTP Injector & HA Tunnel Plus
  • Carrier-specific — adapts to your network
